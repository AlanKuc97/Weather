## Requirements

npm (tested on version 6.14.6)

## How to run (using cmd or terminal)

1. git clone https://github.com/AlanKuc97/Weather.git

2. cd Weather

3. npm install 

4. npm start 

5. Open web-browser on localhost:3000

## Used tools for development

2. Prepros (for SCSS)
